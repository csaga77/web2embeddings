# BoxOccluder3D

Inherits: Occluder3D < Resource < RefCounted < Object

Cuboid shape for use with occlusion culling in OccluderInstance3D.

## Description

BoxOccluder3D stores a cuboid shape that can be used by the engine's occlusion
culling system.

See OccluderInstance3D's documentation for instructions on setting up
occlusion culling.
